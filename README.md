# projiro-legal

Public legal documents for Projiro apps, served via GitHub Pages.

- **Isola privacy policy (IT/EN):** `isola/privacy/index.html`
  → https://ericnsamba.github.io/projiro-legal/isola/privacy/

Future apps get their own folder (`<app>/privacy/index.html`) so every policy
lives at a stable, store-safe URL.
